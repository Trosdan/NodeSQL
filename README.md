## <a id='introducao'></a> Introdução

Neste projeto foi contruido uma APIRest com Express utilizando a ORM Sequelize para fazer conexao com o banco de dados, colocando em pratica conteitos de banco de dados.

### BackEnd

- NodeJS
- ExpressJS
- Sequelize

### Banco de dados

- PostGreSQL
